# CSV File Splitter

A CSV file splitter built using Scala 3. It uses memory-mapped files to read very large files and split them.

# Usage

```sh
./file-splitter --help
file-splitter 0.0.1
Usage: file-splitter [options]

  --input-file <value>
  --output-dir <value>
  --max-lines <value>
  --max-bytes <value>
  --help      prints this usage text
```
